## 🔧 KULLANIM REHBERİ

### 🚀 Hızlı Başlatma
```bash
# Tüm sistemi başlat
python TRM_SYSTEM_STARTER.py
```

### 📊 API Durumu Kontrolü
```bash
# API servislerini kontrol et
python API_INTEGRATION_MANAGER.py --status
```

### 🔑 API Anahtarları Ekleme
`secrets.env` dosyasına yeni anahtarlar ekleyin:
- Telegram: BotFather'dan alın
- OpenAI: OpenAI platformundan alın
- Google Drive: Google Cloud Console'dan alın
- Telegram/Discord/Viber: Telegram/Discord/Viber Business API
- Diğer: İlgili platformların API'leri

### 📋 HTML Panelleri
- **Ana Panel:** `ANA_PANEL.html` - Sistem durumu ve kontrol
- **Status Paneli:** `STATUS_PANEL.html` - API durum takibi
- **Satış Paneli:** `SATIS_PANEL.html` - Komisyon ve gelir takibi
- **Gelişmiş Panel:** `GELISMIS_PANEL.html` - İleri seviye kontroller
- **Drive Sosyal:** `DRIVE_SOCIAL_PANEL.html` - Sosyal medya otomasyonu
- **Cloud Deploy:** `TEK_TIK_CLOUD.html` - Bulut dağıtım
- **Panel Açıcı:** `PANEL_ACICI_HTML.html` - Hızlı erişim
- **Tek Buton:** `TEK_BUTON_YENI.html` - Tüm panelleri bir tıkla aç

### 🌐 API Portları
- **9001:** Ana sistem (durum, kontrol)
- **9002:** Satış sistemi (komisyon, gelir)
- **9003:** Gelişmiş kontroller (AI, optimizasyon)
- **9004:** Drive sosyal medya (otomasyon, paylaşım)
- **9005:** Telegram bildirim (bot, mesajlaşma)
- **9006:** OpenAI entegrasyonu (AI, analiz)
- **9007:** Google Drive (yedekleme, dosya)
